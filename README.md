# 🍛 NutriVision AI - Intelligent Food Recognition & Nutrition Analysis

NutriVision AI is a deep learning–based system that identifies Indian food dishes from images and provides detailed nutritional insights such as calories, protein, fats, and carbohydrates, along with intelligent healthy alternatives.

---

## 🚀 Features

* 🔍 AI-powered food recognition (80+ Indian dishes)
* 📊 Nutrition analysis (Calories, Protein, Carbs, Fats)
* 🥗 Healthy alternative suggestions
* 📈 Interactive charts using Plotly
* 🌐 Streamlit web interface

---

## 🧠 Model Details

* EfficientNet-B3 architecture
* Ensemble of two deep learning models
* Built using PyTorch and Timm

---

## 🛠️ Tech Stack

* PyTorch
